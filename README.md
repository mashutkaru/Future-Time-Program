# Future Time Delegation · Japan

Clean static project prepared for Cursor.

## Files

- `index.html` — app structure
- `styles.css` — visual design and responsive layout
- `app.js` — program data, rendering logic, navigation, and tests

## How to run

Open `index.html` in a browser.

In Cursor, open this folder and edit:
- Schedule/day content in `app.js`
- Visual styling in `styles.css`
- Page structure in `index.html`

## Notes

- The top navigation is sticky.
- The hero section was removed.
- The app includes basic `console.assert` tests in `app.js`.
- Days 7.7, 8.7, and 15.7 show hero images from `assets/`; other days have no photo.
